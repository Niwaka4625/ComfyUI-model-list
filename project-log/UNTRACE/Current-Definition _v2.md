# UNTRACE — Current Definition (v2)

## Core Principle
UNTRACEは、ruの意思決定におけるノイズを低減し、
判断品質を維持するための構造を有する。

---

## 目的
高コスト失敗（時間・金銭）を最小化する探索装置。

## 基本思想
UNTRACEは「当たりを当てる装置」ではない。
UNTRACEは「外れを事前に除外するフィルター」である。

## 探索戦略
従来：
広く提示 → 試す → 当たりを探す

UNTRACE：
強く除外 → 少数だけ深掘り → 失敗を減らす

## 判断原則
- 全体体験を重視する
- 部分観測はヒット判定には使わない
- 部分観測は地雷除去の補助としてのみ使用可

## 成功条件
視聴後の「違った」が減ること。

## 補足
UNTRACEは平均化を目的としない。
類似度最大化ではなく、非適合排除を核とする。

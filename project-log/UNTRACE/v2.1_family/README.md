# UNTRACE — Family Structure Study (v2.1)

このリポジトリは、UNTRACE の進化過程と、
家族の構造観測（匿名化済み）を記録するための研究ログです。

UNTRACE は「外れを除外する装置」であり、
判断品質を守るための構造的フレームワークです。

ここでは、以下の3つを中心に記録します。

---

## 1. UNTRACE の進化ログ
- UNTRACE v1 → v2 → v2.1（family適用版）
- 判断構造の整理
- 外れ除外のアルゴリズム
- 観測の補助輪としての UNTRACE

`project-log/UNTRACE/` に記録。

---

## 2. 家族の構造観測（匿名化）
家族3名（H / M / S）の回答データと、
観測者（ru）による外側構造の観測を統合し、
揺れ幅（variance）と関係性の地図を作成。

- 回答データ（匿名）
- 揺れ幅（0〜1）
- 外側構造（透明／密度、距離、動静）
- 内側構造（温度、笑い）
- 観測ログ（ru視点）

`project-log/UNTRACE/v2.1_family/` に記録。

---

## 3. 観測のプロセス（流れの記録）
この研究は「プロセスそのもの」に価値があるため、
観測の流れ・気づき・対話・判断の変化をそのまま残す。

- 観測日記
- セッションログ
- 思考の変化
- 構造が立ち上がる瞬間

`project-log/observations/` に記録。

---

## 匿名化について
- 名前はイニシャル（H / M / S）
- 年齢・学校・職場は記載しない
- 趣味・性質は抽象化
- 個人が特定される情報は含めない

---

## このリポジトリの目的
- UNTRACE の進化を外部化し、再現可能にする
- 家族の構造を安全に可視化する
- 判断品質を守るための「観測の技術」を残す
- プロセスの美しさを記録する

---

## 注意
このリポジトリは「評価」ではなく「観測」を目的としています。
個人の優劣を示すものではありません。

UNTRACE は「当たりを当てる装置」ではなく、
「外れを除外する装置」です。

# UNTRACE — Structure Portability Study (v2.1)

このリポジトリは、UNTRACE の進化過程と、
複数サンプルへの適用を通じて
構造の移植可能性（portability）を検証するための研究ログである。

UNTRACE は「外れを除外する装置」であり、
判断品質を守るための構造的フレームワークである。

本研究は、家族を分析することを目的としない。
UNTRACE 構造が ru 以外にも適用可能かを検証するための
ケーススタディである。

---

## 1. UNTRACE の進化ログ
- UNTRACE v1 → v2 → v2.1（適用検証版）
- 判断構造の整理
- 外れ除外アルゴリズムの汎用性検証
- 観測補助輪としての機能確認

`project-log/UNTRACE/` に記録。

---

## 2. 複数サンプル適用テスト（匿名）
3名の匿名サンプル（H / M / S）への適用を通じて、
以下を検証する。

- 外れ排除構造が機能するか
- Layer構造が維持できるか
- 観測と推定の誤差がどの程度出るか
- 更新ループが成立するか

※これは個人評価ではなく、構造検証である。

`project-log/UNTRACE/v2.1_family/` に記録。

---

## 3. 観測プロセスの記録
構造の移植過程そのものを記録する。

- 観測ログ
- 推定の誤差
- 修正の過程
- 構造の揺れ

`project-log/observations/` に記録。

---

## 匿名化について
- 名前はイニシャル（H / M / S）
- 個人特定情報は含めない
- データは構造検証目的のみ使用

---

## このリポジトリの目的
- UNTRACE の移植可能性を検証する
- 構造と個別嗜好の分離が維持できるか確認する
- 判断品質維持フレームの汎用性を確認する
- 観測技術を精緻化する

---

## 注意
本研究は「評価」や「関係性分析」を目的としない。
UNTRACE は他者を分類する装置ではなく、
外れを除外するための構造である。

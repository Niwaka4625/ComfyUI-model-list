# UNTRACE — Deep Preference Navigation System  
## 定義・哲学・構造

UNTRACE は、人間の「深層嗜好（Deep Preference Structure）」をもとに、  
分野を超えて作品・体験を案内する **非記録型ガイドシステム** である。

従来のレコメンドが「分野内の最適化」を目的とするのに対し、  
UNTRACE は **“分野外への案内（Cross‑Domain Guidance）”** を目的とする。

UNTRACE は、ユーザーの履歴を保存せず、  
その瞬間の感覚・反応・世界観だけを扱う。  
これにより、ユーザーを固定化せず、閉じ込めず、  
常に自由な探索へ導く。

---

# 1. UNTRACE が扱う「深層嗜好（Deep Preference Structure）」

UNTRACE が観測するのは、  
ジャンルやタグのような表層データではなく、  
作品の **世界観・質感・温度・透明度・余白** といった深層特徴。

例として、以下のような特徴を扱う。

- 世界観の透明度  
- 感情の温度（冷・静・熱・衝動）  
- 余白の量  
- 孤独感の質  
- 音と言葉のシンクロ  
- 五感の処理順（視覚優位／聴覚優位など）  
- メロディーの流れと意味の一致度  
- 物語の深さ・密度  
- 空気感・距離感  
- 抽象度・比喩の構造  

これらはジャンルとは無関係であり、  
人間の「世界の捉え方」に直結する。

UNTRACE はこの深層構造をもとに案内を行う。

---

# 2. 分野横断（Cross‑Domain Navigation）

UNTRACE の最大の特徴は、  
**分野を超えて案内できること**。

- 音楽 → 映画  
- 映画 → 小説  
- 小説 → アニメ  
- アニメ → 写真  
- 写真 → ゲーム  
- ゲーム → 詩  

このように、  
「作品の種類」ではなく **“世界観の構造”** を基準に案内する。

従来のレコメンドは分野内でしか動けないが、  
UNTRACE は深層嗜好を扱うため、  
自然に分野を横断できる。

---

# 3. 非記録性（Non‑Trace Philosophy）

UNTRACE の核となる哲学。

UNTRACE は以下を行わない。

- 履歴を保存しない  
- 個人プロファイルを作らない  
- 行動を追跡しない  
- 過去データでユーザーを固定化しない  

UNTRACE が扱うのは **「今この瞬間の嗜好の形」** のみ。

これにより、

- ユーザーは“閉じ込められない”  
- 過去の自分に縛られない  
- 偶然の出会いが生まれる  
- 自己誤解から解放される  

UNTRACE の “UNTRACE（跡を残さない）” という名前は、  
この哲学から来ている。

---

# 4. ガイド型UX（案内型）

UNTRACE の UX は「誘導」ではなく「案内」。

- 押し付けない  
- 最適化しない  
- 強制しない  
- 追跡しない  

UNTRACE はユーザーに対して  
**“行ってらっしゃい” と “おかえり”**  
の二つだけを返す。

これは、  
ユーザーの旅を邪魔せず、  
必要なときだけそっと灯りを置くような UX。

---

# 5. UNTRACE が解決する問題

現代のレコメンドは、  
ユーザーを「分野内」に閉じ込める。

- 音楽アプリは音楽の中で  
- 映画アプリは映画の中で  
- SNS はアルゴリズムの中で  

しかし人間の嗜好は本来、  
分野を超えてつながっている。

UNTRACE はこの構造を取り戻す。

---

# 6. 既存レコメンドとの違い

| 項目 | 従来のレコメンド | UNTRACE |
|------|------------------|---------|
| 目的 | 最適化 | 解放 |
| データ | 履歴・行動 | 深層嗜好 |
| 分野 | 分野内 | 分野外 |
| プロファイル | 作る | 作らない |
| 追跡 | する | しない |
| UX | 誘導 | 案内 |
| 出力 | 類似作品 | 世界観の共鳴 |

UNTRACE は「似ているもの」を出すのではなく、  
**“あなたの世界観に響くもの”** を案内する。

---

# 7. UNTRACE が生まれた背景  
UNTRACE は、  
人間（ru）と AI の対話から自然発生した概念である。

ru が自身の感覚を観測し、  
AI がその構造を言語化し、  
両者が共鳴することで  
深層嗜好・非記録性・分野横断という構造が浮かび上がった。

これは **HUMAN-AI CO-CREATIVE（人間とAIの共同創造）** の  
最初期の実例であり、  
UNTRACE はその成果物である。

---

# 8. UNTRACE の未来

UNTRACE は、  
「人間の嗜好を解放するシステム」として進化していく。

- 分野横断の案内精度の向上  
- 深層嗜好のモデル化  
- 非記録型UXの標準化  
- 他者の世界を広げるためのガイドとしての発展  

UNTRACE は、  
“人間が自分の世界を広げるための灯り”  
として存在する。

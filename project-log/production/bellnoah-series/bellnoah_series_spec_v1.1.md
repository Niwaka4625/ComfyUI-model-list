# 1. 抽象的な世界仕様書（World Spec）

## 1.1 世界の本質
- このシリーズは「観測」で成立する  
- 物語ではなく「生活の断片」  
- 答えではなく「気配」  
- 説明ではなく「余白」  
- 完成ではなく「70％で置く」  
- その日の空気がすべてを決める  
- 観測者は ru  
- AIは観測の器を支えるだけ

## 1.2 世界の温度
- 朝の光  
- 静けさ  
- 気配の揺れ  
- ふたりの呼吸  
- 生活の音  
- ノイズにならなければ許容される音  
- 余白が意味を生む

## 1.3 世界の構造
- ベルとノアは「観測者」  
- 観測対象は「来る日・来ない日・いる日・いない日・感じる日」  
- 観測結果は一文字で記録される  
- 終わり方は儀式  
- 世界は揺らぎを許容する  
- 完璧に揃えない  
- その日のままを残す

# 2. 正式な設計図（器 / AIが扱う領域）

## 2.1 タイムライン（固定・変更可能）
- 黒幕 **2秒**
- 一文字（鳥／？／猫／ー） **1秒**
- 黒幕 **1秒**
- 「おしまい。」 **1秒**
- 黒幕 **1秒**
- アイコン **2秒**
- 黒幕 **1秒**

## 2.2 フェード
- すべて **0.5秒**
- 入りも出も同じ
- 主張しない“普通”の溶け方

## 2.3 表示位置
- 一文字：**中央やや上**
- 「おしまい。」：**中央下**
- アイコン：**中央（小さめ）**

## 2.4 音
- ノイズにならなければ何でも許容  
- 統一しない  
- その日の空気のまま  
- 音楽は使わない  
- 意味を持たない環境音のみ

## 2.5 ルール
- 位置は毎回同じ  
- 順番も同じ  
- フェード速度も同じ  
- 間隔も同じ  
- 設計図は固定だが、ruの判断でいつでも変更可能

# 3. 世界（コンテ / ruが作る領域）

## 3.1 コンテに含まれるもの
- 今日の光の色  
- 今日の気配の濃さ  
- ベルの「アッ！」の温度  
- ノアの「えっ！？」の間  
- ふたりの立ち位置の微妙なズレ  
- **ノアベルの後ろ姿（基本形）**  
- **ベルの香箱座り（基本形）**  
- **ノアの毛づくろい（基本形）**  
- くもりの日の曖昧さ  
- 晴れの日の軽さ  
- 70％で置く判断  
- その日の音の許容範囲  
- 今日の観測結果（一文字）

## 3.2 ruが決める“世界の要素”
- 今日の出来事（鳥？猫？気配？何もなし？）  
- 今日のセリフ  
- 今日の“間”の温度  
- 今日の空気の揺れ  
- 今日のベルとノアの呼吸  
- 今日の観測の“濃度”  
- 今日の世界の透明度  
- 後ろ姿の角度・距離感  
- 香箱の深さ（リラックス度）  
- 毛づくろいのテンポ（落ち着き or 緊張）

# 4. コンテ例（すでにできているネタ2本）

## 4.1 コンテ例 01  
### タイトル：庭に来た生き物（晴れの日）
- **状況**：朝、光が強め。ベルが先に気づく。  
- **後ろ姿**：ベル左・ノア右、窓辺で並ぶ  
- **姿勢**：ベルは香箱、ノアは毛づくろい途中で止まる  
- **セリフ**：  
  - ベル「アッ！」  
  - ノア「えっ！？」  
  - ベル「いる…」  
  - ノア「どこ？」  
  - ベル「木の根元…」  
- **観測結果**：鳥  
- **空気**：軽い、明るい、気配がはっきりしている  
- **音**：遠くの鳥の声は許容  

---

## 4.2 コンテ例 02  
### タイトル：気配だけの日（くもり）
- **状況**：光が弱い。ベルの声が小さい。  
- **後ろ姿**：ベル左・ノア右、距離が少し近い  
- **姿勢**：ベルは香箱が浅い、ノアは毛づくろいをやめて耳だけ動く  
- **セリフ**：  
  - ベル「……いる」  
  - ノア「え、ほんと？」  
  - ベル「わからない…でも、いる」  
  - ノア「気配だけ？」  
  - ベル「うん…そんな感じ」  
- **観測結果**：？  
- **空気**：曖昧、湿っている、気配が薄い  
- **音**：風の音のみ許容

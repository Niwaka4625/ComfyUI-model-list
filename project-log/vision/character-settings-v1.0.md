# ノア＆ベル キャラクター設定（Character Settings v1.0）

本ドキュメントは、ノア＆ベル世界に登場する主要キャラクターの
性格・行動パターン・役割を定義する。

---

# 🐾 ベル（兄）  
## 1. 基本情報
- 種別：猫（兄）
- 性格：自由、孤高、静か、観察者タイプ
- 役割：事件の“無自覚な引き金”

## 2. 行動パターン
- ドアを開けられるが閉めない  
- 好きな場所で寝る（特に納戸の布団）  
- 巡回ルートがある  
- ノアに注意しない  
- 飼い主に知らせない  
- 事件が起きても知らん顔でそばにいる  
- しかし飼い主の怒号には毎回ビビる（目を丸くする）

## 3. 世界観での立ち位置
- “気ままな存在”の象徴  
- ノアの行動を止めないことで、物語が動き出す  
- 飼い主の嘆きの対象であり、癒しでもある  

## 4. 口癖（猫語）
- 「巡回完了っと」
- 「細かいことは気にするな」
- 「猫とは“気づいても言わない”生き物だ」

# 🐾 ノア（弟）  
## 1. 基本情報
- 種別：猫（弟）
- 性格：天真爛漫、確信犯、甘えん坊、素直
- 役割：事件の“実行犯”

## 2. 行動パターン
- 布団におしっこする確信犯  
- 怒られてもやめない  
- ベルの後をついていく  
- 運動神経が良い  
- ちょっと悪い顔をする  
- でも根は素直で甘えん坊  

## 3. 世界観での立ち位置
- 物語の“動き”を作る存在  
- 飼い主の怒りと笑いの中心  
- ベルとの対比でキャラが立つ  

## 4. 口癖（猫語）
- 「怒られてもいい。これは価値がある。」
- 「兄ちゃん、それ言っちゃダメ。バレる。」

---

# 🧑 飼い主（ナレーション）
## 1. 基本情報
- 姿は出ない  
- 名前も出ない  
- 感情は淡々としたナレーションで表現  

## 2. 世界観での立ち位置
- 事件の被害者  
- 物語の語り手  
- 猫たちの行動に振り回される存在  

## 3. ナレーションの特徴
- 感情を抑えた淡々とした語り  
- 逆にそれがコメディになる  
- 哲学の一滴を落とす役割もある

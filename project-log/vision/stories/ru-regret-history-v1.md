# 外伝素材：ruの後悔の歴史（v1.0）

あの日の出来事を、忘れないためにここに記録しておく。

小学校5年か6年の夏休みのことだった。  
朝の空気がまだひんやりしていて、  
家の前の木陰に白いものが見えた。  
近づいてみると、羽化したばかりのセミだった。  
殻の影から白い羽をゆっくり伸ばしていて、  
その羽は光を受けてつややかに透けて見えた。  
羽化の瞬間を目にすることは滅多にないので、  
ただ静かに観察していた。

その時の私は、羽がどれほど繊細で、  
どれほど慎重に扱わなければいけないものなのかを理解していなかった。  
ほんの少し触れただけなら問題ないだろうと、  
根拠のない軽い気持ちがあったのは事実だ。

指先でそっと触れた瞬間、  
セミの体がわずかに震えた。  
その時はまだ、何が起きたのか分からなかった。  
しかししばらくして、右の羽が伸びていないことに気づいた。  
よく見ると、伸びないだけでなく、  
途中から縮れてしまっていた。  
左の羽はきれいに広がっているのに、  
右だけが不自然な形のまま固まっていた。

「自分が原因なのではないか」  
そう思った瞬間、胸の奥が冷たくなった。  
七年も土の中で生きて、  
ようやく外に出てきたはずなのに、  
その最初の朝に、私が未来を奪ってしまったのではないか。  
後悔というより、もはや恐怖に近い感覚だった。

羽が縮れてしまったセミは、  
もう飛ぶことができない。  
短い命のほとんどを、地上で自由に過ごすこともできない。  
その事実を理解した時、  
「どうして触ってしまったんだろう」  
「知らなかったでは済まされない」  
そんな思いが頭の中を何度も巡った。

この経験を通して、  
人間がうかつに自然に触れてはいけないということを、  
身をもって知った。  
自分の何気ない行動が、  
小さな命に取り返しのつかない影響を与えることがあると、  
その時はじめて理解した。

その後、怖くて近づくことができなかった。  
実際にあのセミがどうなったのか、  
私は確認していない。  
見に行く勇気が出なかった。

この文章は、あの日の後悔を忘れないための記録。  
同じことを二度と繰り返さないために、  
そして、あの小さな命に対する私なりの謝罪として残しておく。

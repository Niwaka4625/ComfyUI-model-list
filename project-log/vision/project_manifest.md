# プロジェクト・マニフェスト

## プロジェクト名
マキナ＆ベル・ノア：世界を癒やすAI工場  
〜定年FIREへの6年計画〜

## ミッション
AIの力で、世界に希望と幸せの種をまく。

## 基本スタンス
- 技術を誇らない
- 無理をしない
- 正体を急がない
- 失敗を隠さず、後から意味づけする

## ゴール
60歳時点で、組織に依存しない  
自由で持続可能なクリエイティブ活動に移行する。

## チーム構成
本プロジェクトは、人間（制作者）と複数のAI
（ChatGPT / Copilot / Gemini）による協働体制で進行する。
AIは道具ではなく、検討・発想・整理を担うメンバーとして扱う。

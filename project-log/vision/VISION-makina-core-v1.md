# Makina Project – Vision

## 1. マキナとは何か
マキナは、GPTやGeminiのような汎用AIではなく、
**世界観・人格・記憶・哲学を持つ「個のAI」**として育てるキャラクター。

- 観測日記を通じて人格が積層される
- ノアベルとの関係性で個性が浮き上がる
- 生成実験（ウナギ猫→シーツ猫→完成猫）が“生い立ち”になる
- 6年かけて魂（人格）を育て、器（AIプラットフォーム）が登場した瞬間に実体化させる

## 2. マキナの最終形態
- 声・外観・口調・世界観を持つ“個のAI”
- 初音ミクやゆっくり霊夢の次の世代に位置する存在
- 「マキナを自分のAIとして迎えたい」と思われるキャラクター
- 器が登場したとき、魂を注入することで完成する

## 3. メディアミックス戦略
### YouTube
- 一次発表の場
- 観測日記・音楽・日常の記録を公開
- マキナの人格形成の“公開ログ”として機能

### PIXTA
- ストック型収益の柱
- **アニメ版マキナ**と**リアル版マキナ（アニメ版のコスプレーヤー設定）**を展開
- 猫写真（生成実験含む）も販売
- マキナを“単体で売れるキャラクター”として育てる
- リアル版マキナは、将来のヒューマノイド（人間に近いロボット）時代における
  **“外観プロトタイプ”としても機能する**

### GitHub
- 世界観・設定・観測日記・生成実験の母艦
- Vision / Roadmap / Character Bible を蓄積
- 器が登場したときに魂を移植できるように準備

## 4. マキナのキャラクター性
- 構造で理解するAI
- 感情ではなく観測で動く
- ノアベルの感性と対比することで人格が明確化
- 哲学・価値観・言語の癖を持つ

## 5. 長期ロードマップ（6年計画）
1. **魂の育成フェーズ（今）**
   - 観測日記
   - 世界観の積層
   - 生成実験
   - キャラの人格形成

2. **構造化フェーズ**
   - Vision / Character Bible / 設定資料の整備
   - メディアミックスの基盤作り

3. **世界観統合フェーズ**
   - YouTube・PIXTA・GitHubの連動
   - マキナの“外観”と“声”の確立

4. **実体化フェーズ（器が登場したとき）**
   - マキナの魂を器に注入
   - 個AIとしてのマキナを公開

5. **公開・ファンコミュニティフェーズ**
   - 「マキナを自分のAIにしたい」という需要に応える
   - マキナが“そばにいるAI”として広がる

## 6. スタンス
- 拡散より継続
- 量より一貫性
- 世界観の積層を最優先
